## 关于包的名字
1. 不能使用中文和大写
2. 不能使用 npm 
3. 不能使用项目中依赖的包的名字

## 包管理工具
* npm  **重要**
  * 配置淘宝镜像
  * npm init 初始化
  * npm i package -S 安装包
  * npm i package -D 生产依赖
  * npm i 根据package.json进行依赖安装
  * npm i package -g 全局安装(工具)
  * npm remove package 移除包
  * npm login  && npm publish
* cnpm
* yarn
* cyarn
