## 介绍
Markdown 是一种可以使用普通文本编辑器编写的标记语言 

## 作用
1. 写笔记
2. 写文档
3. 写博客

## 特点
1. 语法简洁
2. 样式和内容分离
3. 编辑方便

## 编辑使用的软件
* typora								<https://www.typora.io/>
* 有道云笔记                        <http://note.youdao.com/>
* markdownpad
* webstorm  HBuilder  vscode

## 语法
#### 兼容 HTML

```
<span style="color:red">兼容html哦</span>
```

#### 标题

```
## 标题
### 标题
#### 标题
```

#### 引用

```
> 我的舞蹈没人能跳第二遍,包括我自己
```

#### 无序列表

```
* 苹果
* 香蕉
* 梨子
- 菠萝
- 山楂
- 桔子
```

#### 有序列表

```
1. apple
2. 华硕
3. 宏基
```

#### 代码块儿

```
​```js
​```
```

#### 分割线

```
---
```

#### 加粗

```
**他不是猪才怪**
```

#### 小段代码 加亮

```js
`console.log('iloveyou')`
```

```
